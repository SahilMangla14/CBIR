The project has a frontend and a backend

Frontend serves as a input for the query image, and displays the output based on a selected method
Backend runs the appropriate functions to achieve result of any particular method

To run the frontend/website, run the commands:
    cd frontend
    npm install
    npm start
To run the python files in backend, install the following libraries:
    open-cv
    numpy
    sklearn
    torch, torchvision
    tensorflow
    flask, flask_cors

To run the server, run the commands:
    cd backend
    python3 server.py
